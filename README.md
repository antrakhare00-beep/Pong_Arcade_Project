## 🏓 Pong Game

A classic Pong game built using HTML5 Canvas and JavaScript as part of my web development learning journey.

## 📖 Project Background

This project was developed while learning JavaScript and HTML5 Canvas. As a beginner, I referred to online learning resources, documentation, and AI-assisted guidance to understand game development concepts and implement various features.

Rather than simply copying code, I focused on understanding how each component works, experimenting with the logic, fixing bugs, and making modifications to improve my learning experience.

## 🚀 What I Learned

Through building this project, I gained practical experience with:

- JavaScript fundamentals
- HTML5 Canvas rendering
- Collision detection
- Game loops
- Event handling
- Score tracking systems
- Responsive design principles
- Debugging and problem-solving

## 🎮 Features

- Two-player Pong gameplay
- Paddle controls
- Ball movement and collision detection
- Score tracking
- Winning screen
- Center dashed line
- Responsive fullscreen canvas

## 🛠️ Technologies Used

- HTML5
- JavaScript
- Canvas API

## 🌱 Learning Note

This project is part of my journey as a beginner developer. While I used external resources and guidance to help me understand concepts, I actively worked through the implementation, debugging process, and customization to strengthen my programming skills.
